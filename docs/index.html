What is Celo?

Celo is an open platform that makes financial tools accessible to anyone with a mobile phone

See our website or read the docs for more info!

What is Celo Blockchain?

Celo Blockchain is the primary implementaion of the Celo protocol. It is based on Geth, the ethereum client, and it will be refered to as "geth" throught the docs.

See our repository for the code!
